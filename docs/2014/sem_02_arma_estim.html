Оценка ARMA моделей в R

Подключаем пакет для оценки гарчей в R и прочие удобства:

library("lubridate")
library("ggplot2")
library("dplyr")
library("xts")
library("quantmod")
library("forecast")

Оцениваем модель ARIMA(1,1,1): \[ \triangle y_t = \phi_1 \triangle y_{t-1} + \theta_1 \varepsilon_{t-1} + \varepsilon_t \]

model <- Arima(y, order=c(1,1,1))
summary(model)

Можно попросить R самостоятельно подобрать модель руководствуясь (упрощенно) информационным критерием:

auto_model <- auto.arima(y)
summary(auto_model)
